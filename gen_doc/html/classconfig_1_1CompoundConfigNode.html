template<typename T >
template bool config::CompoundConfigNode::setScalar ( const T  value)

Scalar setter (template).

Sets the node to a Scalar value.

This is made in a template format for standardization across all Scalar types to reduce the amount of code that needs to be changed upon refactor. In order to avoid linker issues, please add an explicit instantiation at the bottom of the file in order to avoid linker issues.

Template Parameters
TThe C++ type of the Scalar we wish to set.
Parameters
scalarThe Scalar we wish to set.
Returns
Whether or not the scalar we wanted to set was set.
Postcondition
If return is true the set was successful. If return is false, the value at the node was not replaced.
